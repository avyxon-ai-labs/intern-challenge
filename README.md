# Avyxon AI Labs — Flutter Internship 1-Pager

A single-file HTML page for the Avyxon AI Labs Flutter Internship Challenge 2026.

## File

```
avyxon_2026_final.html
```

Open directly in any browser. No build step, no dependencies, no server needed.

## What's Inside

- All content, styles, and animations in one self-contained `.html` file
- Logo embedded as base64 — works fully offline
- Scroll-triggered section reveals and hover micro-interactions
- Responsive down to mobile

## Fonts Used

- **Bricolage Grotesque** — headings
- **JetBrains Mono** — labels, tags, code
- **Lato** — body text

Loaded via Google Fonts CDN. Requires internet on first load; cached after.

## Customisation

| What | Where |
|---|---|
| Contact email | Search `info@avyxon.ai` — appears 4 times |
| Year | Search `2026` — appears in nav, hero, footer |
| Logo | Replace the base64 `src` on the `<img>` tags |
| Accent colour | `--accent` in `:root` CSS variables |
